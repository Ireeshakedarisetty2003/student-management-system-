## Student Management System

##  Project Overview
The Student Management System is a database-driven application developed using Python and SQL (SQLite). It helps in managing student records efficiently by automating data storage, retrieval, and analysis.

---

##  Features
- Add new student records
- View student details
- Update student information
- Delete student records
- Perform data analysis on student performance
- Ensure data validation and accuracy

---

##  Technologies Used
- Python
- SQL (SQLite)
- Pandas (for data analysis)

---

##  Project Description
This project focuses on building a structured system to handle student data digitally. It replaces manual data handling with an automated system, improving efficiency and reducing errors.

The system performs CRUD operations:
- **Create** – Add new student data
- **Read** – Retrieve and display data
- **Update** – Modify existing records
- **Delete** – Remove unwanted data

Python is used for backend logic, while SQLite is used for database management. Pandas is used to analyze student performance such as average scores and top-performing students.

---

##  How It Works
1. User inputs student data
2. Data is stored in SQLite database
3. Python executes SQL queries
4. Data is retrieved and displayed when required
5. Analysis is performed using Pandas
